# RAG Knowledge Assistant

This project implements a retrieval-augmented generation (RAG) system that answers user queries based on custom documents.

## What it does
- Ingests and processes text documents
- Converts them into embeddings
- Stores embeddings in a vector database
- Retrieves relevant context for a query
- Uses an LLM to generate answers grounded in retrieved data

## Why I built this
I wanted to understand how LLM systems use external knowledge instead of relying only on pre-trained data.

## Tech
Python, OpenAI API, ChromaDB
